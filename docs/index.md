# Rock Paper Scissors CI/CD Project

Welcome to the official documentation for the Rock Paper Scissors university project. This project focuses on modern software engineering practices, including automated testing and CI/CD pipelines.

## Project Objectives
- **Functional Mini Application**: A robust RPS engine with "Best of N" logic.
- **CI/CD Pipeline**: Fully automated testing and deployment via GitHub Actions.
- **Automated Testing**: Comprehensive Whitebox and Blackbox testing suites.
- **Professional Documentation**: Utilizing MkDocs for high-quality technical documentation.

## Artifact Under Study
The core of this project is `game.js`, a JavaScript module that manages the game state, validates moves, and determines winners based on a configurable round count.
